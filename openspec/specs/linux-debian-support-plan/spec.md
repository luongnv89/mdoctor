# linux-debian-support-plan Specification Delta

### Requirement: Debian-only support roadmap
- The project SHALL define a phased roadmap for Debian-based Linux support.

#### Scenario: Maintainer planning
- GIVEN a maintainer preparing Linux support work
- WHEN reviewing roadmap docs
- THEN they can execute a sequenced task plan with clear scope and non-goals.

### Requirement: Explicit initial platform scope
- The roadmap SHALL clearly state initial Linux support is Debian-based only.

#### Scenario: User expectations
- GIVEN a user on non-Debian Linux
- WHEN reading roadmap notes
- THEN they understand support is planned later, not in initial rollout.
